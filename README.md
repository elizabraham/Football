## Football Post Match Analysis Dashboard

Built using Streamlit and mplsoccer to analyze match-level data.

Key Features:
- Shot maps
- Passing networks
- xG visualization
- Tactical zones

Tech Stack:
Python, Pandas, Streamlit, mplsoccer, Matplotlib
